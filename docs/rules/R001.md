# R001 — Fuel Cross-Feed Mismatch

## Real-World Failure Mode

In multi-engine aircraft, pilots manage fuel selectors to balance fuel draw across tanks. A common and dangerous misconfiguration is both engines drawing from the same tank while the other tank(s) still contain usable fuel. This creates an asymmetric fuel exhaustion scenario: the shared tank runs dry first, potentially causing simultaneous engine power loss—even though the aircraft has hundreds of pounds of fuel remaining in the other tank.

This is the failure mode in the origin story for Flight Guardian. In the real-world accident, a Beechcraft Baron suffered dual engine fuel starvation because both engines were feeding from the same tank for an extended period.

A related condition is **sensor/selector disagreement**: fuel flow is detected on an engine whose fuel selector reads as OFF or pointed to a different tank. This may indicate a faulty selector position sensor, a stuck valve, or a fuel system malfunction.

## How the Rule Works

R001 evaluates each engine's fuel selector position and fuel flow rate. It identifies:

1. **Cross-feed condition**: Two or more engines with selectors pointing to the same tank, while at least one other tank holds >1.0 gallon of usable fuel.
2. **Time-to-exhaustion**: Combined fuel flow of all engines on the shared tank divided into the tank's remaining quantity.
3. **Selector disagreement**: Fuel flow >0.5 GPH on an engine whose selector reads as OFF (value <0.5).

## Severity Levels

| Severity | Condition |
|----------|-----------|
| WARNING | Both engines on same tank, >30 minutes to exhaustion |
| CRITICAL | Both engines on same tank, ≤30 minutes to exhaustion |
| CRITICAL (urgent) | Both engines on same tank, ≤15 minutes to exhaustion |
| WARNING | Fuel flow on engine with selector OFF |

## Applicability

- Multi-engine aircraft only (profile `engine_count` >= 2)
- Engines must be running (combustion active)
- All phases

## Key Thresholds

- Time-to-exhaustion critical: 30 minutes
- Time-to-exhaustion urgent: 15 minutes
- Other tank minimum fuel: >1.0 gallon
- Fuel flow threshold for sensor check: >0.5 GPH
- Selector OFF value: <0.5

## Required SimVars

- `FuelTankSelector` (per engine)
- `GeneralEngFuelFlow` (per engine)
- `GeneralEngCombustion` (per engine)
- `FuelSystemTankQuantity` (per tank)
- `FuelTotalQuantity`

## Evaluation Interval

5 seconds

## Limitations

- Relies on SimConnect fuel selector position accuracy, which may not reflect all third-party fuel system implementations
- Time-to-exhaustion is a linear projection based on current flow rate; does not account for power changes
- Cannot detect physical fuel line blockages that SimConnect doesn't model
