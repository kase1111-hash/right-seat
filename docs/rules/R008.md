# R008 — Vacuum System Failure

## Real-World Failure Mode

Many GA aircraft rely on vacuum-driven gyroscopic instruments (attitude indicator, heading indicator) for instrument flight. Vacuum pump failure causes these instruments to slowly spool down, giving increasingly erroneous readings. This insidious failure has caused many fatal accidents in IMC, as pilots unknowingly trust failing instruments. The danger is significantly reduced in VMC where visual references are available.

## How the Rule Works

R008 monitors vacuum system suction pressure and adjusts severity based on flight conditions:

1. **Below minimum suction**: If vacuum suction drops below the profile's minimum threshold, instruments are unreliable. In IMC (cloud or precipitation), this is a WARNING. In VMC, it's downgraded to ADVISORY since the pilot has visual references.
2. **Below normal range**: Suction below the normal operating range (but above minimum) fires ADVISORY as an early warning.

## Severity Levels

| Severity | Condition |
|----------|-----------|
| WARNING | Suction below minimum in IMC |
| ADVISORY | Suction below minimum in VMC (downgraded) |
| ADVISORY | Suction below normal range |

## Applicability

- Airborne only (not applicable in Ground or Landing phases)
- Not applicable to aircraft without vacuum systems (DA62 has suction_minimum_inhg = 0)

## Key Thresholds

- Suction minimum: `profile.vacuum.suction_minimum_inhg`
- Suction normal range: `profile.vacuum.suction_normal_inhg` [low, high]
- VMC context: not in cloud AND no precipitation

## Required SimVars

- `SuctionPressure`
- `AmbientInCloud`
- `AmbientPrecipState`

## Evaluation Interval

10 seconds

## Limitations

- MSFS may not accurately model vacuum pump degradation (partial failures with fluctuating suction)
- VMC/IMC determination is simplified (cloud + precipitation); does not account for night VFR where gyro loss is more dangerous
- Aircraft with glass cockpits and electric gyros may still report vacuum suction from a backup system
